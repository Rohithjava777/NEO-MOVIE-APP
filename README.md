## ✨ About The Project
![Project Overview](https://images.unsplash.com/photo-1489599849927-2ee91cede3ba?q=80&w=1280&auto=format&fit=crop)

**Neo Movie App** is not just another movie database application. It is a highly opinionated, visually striking platform designed to captivate your attention. Blending the unapologetic, bold aesthetic of modern **Neo-Brutalism** with an immersive dark-purple color palette and neon-pink glowing accents, this app is formulated to stand out in any portfolio. 

### 🚀 Key Features
- **Striking UI/UX**: Deep, dark aesthetic perfectly contrasted with sharp borders and neon-pink glowing text drops. Designed to make elements pop off the screen.
- **Micro-interactions:** Silky smooth, heavy frame-animations powered by **Framer Motion**.
- **Real-time Search:** Effortlessly filter moving items instantaneously.
- **Client-Side Routing**: Handled seamlessly, ready for fast transitions without reload.
- **Vercel Ready**: Pre-configured setup for high-speed automated deployment worldwide.

## 🚦 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites
Make sure you have Node.js installed on your machine.
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation
1. Clone the repository
   ```sh
   git clone https://github.com/Rohithjava777/NEO-MOVIE-APP.git
   ```
2. Navigate into the project directory
   ```sh
   cd NEO-MOVIE-APP
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Start the development server
   ```sh
   npm run dev
   ```

## 🎨 Architecture & Design
The architectural aim was to marry bleeding-edge performance with aggressive, premium looks. The custom Tailwand configurations target specific variables directly tailored for this build:
* **Backgrounds:** `bg-[#0B001A]` (A visceral dark purple)
* **Highlights:** `accentNeon: #FF2A93` (High glow pink effect on active elements)
* **Interactivity:** Elements pop drastically on hover via `transition-all duration-300`, creating physical-feeling "pushable" buttons.
